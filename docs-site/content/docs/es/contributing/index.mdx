# Contribuir

Guía completa en la raíz del repositorio: **[CONTRIBUTING.md](https://github.com/LaProgrammerie/hyper-fast-builder/blob/main/CONTRIBUTING.md)**.

## Lista rápida

1. Leer `AGENTS.md` y `docs/ai/context-map.md`
2. Alinear características con `docs/ai/active/handoff.md` y specs de producto (`spec-postv123.md`, etc.)
3. `make build && go test -race ./...`
4. Actualizar la documentación CLI cuando cambien comandos:

```bash
go run ./application/cmd/agentflow docs generate-cli \
  --output docs-site/content/docs/en/cli/generated
```

5. `cd docs-site && npm run build` para cambios MDX
6. Un PR por tema coherente; sin secretos en commits

## Documentación

- Sitio público: `docs-site/content/docs/` (Fumadocs, varios idiomas)
- Canon del proyecto: `docs/ai/`
- Deriva código/doc: [`problems.md`](https://github.com/LaProgrammerie/hyper-fast-builder/blob/main/problems.md) en la raíz

## Decisiones de arquitectura

Registrar compromisos duraderos en `docs/ai/05-decisions.md` y `docs/decisions/`.
