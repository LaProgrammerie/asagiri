```bash
asa dashboard
```

## Widgets

- Runtime
- Agents
- Trust
- Costs
- Flow
- Events
- Progress

## Comportamiento responsive

- Compact (<100 columnas): conjunto reducido en una sola columna.
- Wide (>=120 columnas): layout de dos columnas.
- Ultra-wide (>=180 columnas): layout de tres columnas.

El refresco está limitado para evitar flicker y exceso de CPU.
