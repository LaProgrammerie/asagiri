# Soporte de ratón

Activación:

```yaml
ui:
  mouse: true
```

## Interacciones (lote 7D)

- **Clic izquierdo** — seleccionar fila (explorers, acciones recomendadas en Mission Control)
- **Doble clic** — abrir detalle de nodo/paso o saltar desde eventos
- **Clic derecho** — menú contextual (explain, reanudar graph, pipeline prototype, verify trust…)
- **Hover** — resaltar fila si el terminal soporta cell motion
- **Rueda / arrastrar divisor** — redimensionar paneles split

El teclado sigue siendo obligatorio: cada acción también está en atajos y **Ctrl+P**.
