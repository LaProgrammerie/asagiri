# Soporte de ratón

Activación:

```yaml
ui:
  mouse: true
```

## Interacciones actuales

- eventos de puntero en terminales compatibles;
- resize de split con rueda (básico);
- drag del divisor para resize de paneles (básico).

Todas las acciones críticas siguen disponibles por teclado.
