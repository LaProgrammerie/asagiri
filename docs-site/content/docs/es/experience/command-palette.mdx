# Command Palette

Atajo global:

```text
Ctrl+P
```

## Capacidades

- búsqueda rápida de acciones;
- navegación entre pantallas clave;
- ejecución de acciones vía CommandBus;
- equivalentes CLI visibles para reproducibilidad.

## Safety UX

Las acciones destructivas muestran confirmación con impacto, comando CLI, nota de rollback y confirmación explícita.
