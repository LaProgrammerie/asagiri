# Atajos de teclado

- `Ctrl+P` command palette
- `Ctrl+M` mission control
- `Ctrl+D` dashboard
- `Ctrl+G` graph explorer
- `Ctrl+F` flow explorer
- `Ctrl+K` knowledge explorer
- `Ctrl+T` trust explorer
- `Ctrl+E` explain
- `Ctrl+R` replay explorer
- `Ctrl+L` logs
- `Tab` / `Shift+Tab` foco panel siguiente/anterior
- `?` pantalla de ayuda accesibilidad y teclas
- `Esc` cerrar modal o palette
- `q` salir

El teclado sigue siendo la interfaz principal. El ratón es opcional.
