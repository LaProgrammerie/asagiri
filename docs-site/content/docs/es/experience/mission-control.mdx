# Mission Control

Abrir Mission Control:

```bash
asa
# o
asa mission
```

## Contenido principal

- workspace y rama
- estado runtime y sesión
- pasos del flow activo con glifos de progreso
- resumen Agent Theatre
- dimensiones trust y puntuación global
- **coste día / mes** (derivado de eventos runtime)
- eventos recientes (feed en vivo)
- **acciones recomendadas** — siguientes pasos priorizados con equivalente CLI (`GetRecommendedActionsQuery`)

Usar acciones recomendadas o **Ctrl+P** para work, verify trust, reanudar graph bloqueado o avanzar el pipeline prototype.
