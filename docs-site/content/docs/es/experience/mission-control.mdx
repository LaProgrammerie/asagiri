# Mission Control

Abrir Mission Control:

```bash
asa
# o
asa mission
```

## Información mostrada

- workspace y rama;
- estado de runtime y sesión;
- progreso del flujo activo;
- agentes activos;
- dimensiones de trust y warnings;
- eventos recientes.

Mission Control es el punto de entrada antes de abrir un explorer específico.
