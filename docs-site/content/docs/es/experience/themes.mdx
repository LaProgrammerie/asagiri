# Temas

Temas soportados:

- `asagiri-dark`
- `asagiri-light`
- `minimal`
- `high-contrast`
- `cyber`

Configuración:

```yaml
ui:
  theme: asagiri-dark
```

## Recomendación de accesibilidad

Usa `high-contrast` cuando necesitas máxima legibilidad o trabajas en terminales con bajo contraste.
