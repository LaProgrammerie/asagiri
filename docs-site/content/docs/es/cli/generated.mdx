# Referencia CLI generada

Estas páginas se **generan** a partir de las definiciones Cobra activas. No las edite a mano — se sobrescribirán.

## Regenerar

```bash
go run ./application/cmd/agentflow docs generate-cli \
  --output docs-site/content/docs/en/cli/generated
```

O con un binario compilado:

```bash
./bin/agentflow docs generate-cli --output docs-site/content/docs/en/cli/generated
```

## Idiomas

- **Referencia completa (inglés):** [CLI generado — inglés](/docs/cli/generated) — todos los comandos, todas las banderas.
- Este bloque **español** ofrece páginas resumidas solo para los comandos principales; el detalle de opciones sigue en la versión inglesa generada.

Explore la barra lateral **CLI → generado** o comience en el [índice generado (EN)](/docs/cli/generated/index).
