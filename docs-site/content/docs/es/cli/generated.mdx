# Referencia CLI generada

Las páginas de esta sección se materializan exclusivamente mediante **`docs generate-cli`**. Trate las ediciones manuales en Markdown como efímeras: la siguiente advertencia sigue siendo cierta porque la salida regenerada de Cobra sobrescribe retoques humanos sueltos.

Estas páginas están **generadas** a partir de las definiciones vivas de comandos Cobra. No las edite a mano: los cambios se perderán.

## Regenerar

```bash
go run ./application/cmd/asa docs generate-cli \
  --output docs-site/content/docs/en/cli/generated
```

O con un binario compilado:

```bash
./bin/asa docs generate-cli --output docs-site/content/docs/en/cli/generated
```

## Explorar comandos

Vea la sección **CLI → generado** en la barra lateral, o comience en el [índice generado](/docs/es/cli/generated/index).

En este idioma solo hay resúmenes para un subconjunto de comandos; el conjunto completo de banderas permanece en la referencia en inglés bajo `/docs/cli/generated` (misma generación de código fuente).
