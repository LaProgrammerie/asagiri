# `asa ready`

Alias de `asa onboard --check-only`. Devuelve informe readiness JSON o plain.

## Usage

```bash
asa ready
asa ready --json
asa ready --plain --strict
```

Informe persistido en `.asagiri/onboarding/report.json`.
