# Comandos contracts

`asa contracts extract` transforma flujos en contratos técnicos utilizables.

## Extraer

```bash
asa contracts extract workspace-saas
```

Genera:

- `contracts/api.openapi.yaml`
- `contracts/permissions.yaml`
- `contracts/events.yaml`
- `contracts/analytics.yaml`
- `contracts/observability.yaml`

El reporte de extracción marca incertidumbres para revisar antes de implementar.
