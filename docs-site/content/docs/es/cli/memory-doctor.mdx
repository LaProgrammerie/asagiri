# `asa memory doctor`

Ejecuta controles de salud de la memoria (fase final **PF-A-01**): alcance del embedder, coherencia de dimensiones, entradas huérfanas.

## Uso

```bash
asa memory doctor
```

## Comprobaciones

| Control | Cuándo |
| --- | --- |
| **ollama** | `runtime.memory.embedder: ollama` — API alcanzable |
| **dimensions** | Vectores almacenados = tamaño de salida del embedder |
| **orphans** | Entradas sin metadatos de embedding válidos |
