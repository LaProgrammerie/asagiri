# Comandos flows

`asa flows` deriva recorridos de usuario estructurados desde el prototipo.

## Extraer

```bash
asa flows extract workspace-saas
```

Genera:

- `flows/*.flow.yaml`
- `screens/*.screen.yaml`
- `extraction/extraction-report.md`

## Inspeccionar

```bash
asa flows inspect workspace-saas
asa flows inspect workspace-saas --flow onboarding
```

Muestra cantidad de pasos/acciones, contratos no resueltos y nivel de riesgo.

## Revisar flujos

```bash
asa flows review workspace-saas
```

Ejecuta una revisión orientada al negocio: alineación con objetivo, cobertura de métricas, brechas de observabilidad/seguridad y riesgo de implementación.
