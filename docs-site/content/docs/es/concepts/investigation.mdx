# Motor de investigación

Responde **qué, dónde, por qué** — antes de ejecutar un agente de implementación (`spec-my-A` §25).

## Pipeline

Síntoma → alcance → descubrimiento local → hipótesis → context-pack.md + graph.json → trabajo dirigido.

## Integración

`asa work --investigate-first`, `asa verify --investigate-on-failure`, `asa investigate impact`.
