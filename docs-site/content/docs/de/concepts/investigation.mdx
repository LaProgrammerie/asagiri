# Investigation Engine

Beantwortet **was, wo, warum** — bevor ein Implementierungs-Agent läuft (`spec-my-A` §25).

## Pipeline

Symptom → Scope → lokale Discovery → Hypothesen → context-pack.md + graph.json → gezielte Arbeit.

## Integration

`asa work --investigate-first`, `asa verify --investigate-on-failure`, `asa investigate impact`.
