# Persistente Runtime

Asagiri wird von einer stateless CLI zu einer **local-first Engineering-Runtime** (`spec-my-A` §24).

## Architektur

```text
asa daemon run → SQLite → sessions · events · memory · hooks
```

## Prinzipien

Optional, inspizierbar, sicher (nur `asa …` in Hooks), scoped Memory.

## Modi

```yaml
runtime:
  mode: guided  # guided | interactive | headless | ci | review | exploration
```

## Memory-Embeddings

Scoped Memory mit `runtime.memory.embedder`: `hash` (Standard), `ollama`, `cloud` (nur bei `cloud.enabled: true`). Nach Wechsel: `asa memory reindex`.
