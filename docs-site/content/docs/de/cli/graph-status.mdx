# `asa graph status`

Zeigt den aktuellen Status eines unter `.asagiri/graphs/<graph-id>/` gespeicherten Graphs (`spec-my-C` §5.4).

## Verwendung

```bash
asa graph status graph-2026-05-29-9307e4be
asa graph status graph-2026-05-29-9307e4be --json
```

## Ausgabe

Terminal: Graph-ID, Produkt, Flow, Status, Knoten-/Kanten-/Checkpoint-Anzahl, geschätzte Kosten, höchstes Risiko.

Mit `--json`: vollständiges Graph-Dokument plus Schätzung.
