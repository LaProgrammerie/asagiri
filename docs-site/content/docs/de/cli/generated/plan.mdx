# plan

CLI: **`agentflow plan`**

## Wann einsetzen

Workflow-Befehle treiben verfolgte Features durch spec → plan → Implementierung → Verifikation → Reporting-Schranken vor dem Merge-Request.

## Fehlermodi

- Cobra lehnt unbekannte Flags oder Arity-Fehler ab, bevor Repo-Seiteneffekte entstehen.
- AgentFlow kann Bootstrap nicht auflösen, wenn der Befehl außerhalb eines initialisierten Checkouts läuft.
- Delegierte Subprozesse (**git**, MCP-Hosts, Editor-Agenten usw.) können non-zero exitieren und Fehler 1:1 durchreichen.
