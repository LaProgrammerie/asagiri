# Generierte CLI-Referenz

Die **vollständigen** Seiten (alle Befehle, mit Cobra synchronisiert) werden auf **Englisch** unter [`/docs/cli/generated`](/docs/cli/generated) gepflegt.

Dieses Verzeichnis enthält **übersetzte Kurzfassungen** für die wichtigsten Befehle: Abschnitte „Wann einsetzen“ und „Fehlermodi“, plus Link zur englischen Detailseite.

## Dokumentation neu generieren (Contributors)

```bash
go run ./application/cmd/agentflow docs generate-cli \
  --output docs-site/content/docs/en/cli/generated
```
