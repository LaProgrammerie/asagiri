# contracts Befehle

`asa contracts extract` transformiert Flows in umsetzbare technische Vertraege.

## Extrahieren

```bash
asa contracts extract workspace-saas
```

Erzeugt:

- `contracts/api.openapi.yaml`
- `contracts/permissions.yaml`
- `contracts/events.yaml`
- `contracts/analytics.yaml`
- `contracts/observability.yaml`

Der Extraktionsreport markiert Unsicherheiten vor der Implementierung.
