# `asa ready`

Alias für `asa onboard --check-only`. Liefert JSON- oder Plain-Readiness-Report.

## Usage

```bash
asa ready
asa ready --json
asa ready --plain --strict
```

Report wird in `.asagiri/onboarding/report.json` gespeichert.
