# `asa graph resume`

Setzt einen pausierten, fehlgeschlagenen oder unterbrochenen Graph anhand persistierter Checkpoints fort (`spec-my-C` §5.5, §15).

## Verwendung

```bash
asa graph resume graph-2026-05-29-9307e4be
asa graph resume graph-2026-05-29-9307e4be --json
```

## Verhalten

Lädt `execution-graph.yaml` aus `.asagiri/graphs/<graph-id>/`, stellt Knotenstatus aus Checkpoints wieder her und führt verbleibende ready-Knoten aus. Emittiert `graph.*`-Runtime-Ereignisse während Resume.
