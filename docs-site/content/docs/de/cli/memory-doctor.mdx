# `asa memory doctor`

Führt Gesundheitschecks der Laufzeit-Memory aus (Phase finale **PF-A-01**): Embedder-Erreichbarkeit, Dimensionskonsistenz, verwaiste Einträge.

## Verwendung

```bash
asa memory doctor
```

## Prüfungen

| Prüfung | Wann |
| --- | --- |
| **ollama** | `runtime.memory.embedder: ollama` — API erreichbar |
| **dimensions** | Gespeicherte Vektoren = Embedder-Ausgabegröße |
| **orphans** | Einträge ohne gültige Embedding-Metadaten |
