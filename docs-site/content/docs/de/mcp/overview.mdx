# MCP-Überblick

AgentFlow kann einen minimalen **stdio JSON-RPC**-MCP-Server (`application/internal/mcp`) für Editor-Integrationen bereitstellen.

<Callout type="experimental">
MCP ist standardmäßig deaktiviert. Nur in vertrauenswürdigen Umgebungen aktivieren nach Prüfung der Tool-Scopes.
</Callout>

## Aktivieren

```yaml
mcp:
  enabled: true
  max_output_bytes: 1048576
  command_timeout_seconds: 120
```

## Servieren

```bash
agentflow mcp-serve
```

Unterstützt `initialize`, `tools/list`, `tools/call` (newline-delimited JSON).

## CLI-Wrapper

```bash
agentflow mcp
```

MCP-bezogene Befehle inspizieren — siehe generierte Seiten [mcp](/docs/cli/generated/mcp) und [mcp-serve](/docs/cli/generated/mcp-serve).
