# Roadmap

Kanonische Roadmap: **[ROADMAP.md](https://github.com/LaProgrammerie/hyper-fast-builder/blob/main/ROADMAP.md)** im Repository-Root.

## Geliefert (V1 → V3)

- CLI-Primitives: `init`, `spec`, `plan`, `dev`, `verify`, `review`, Worktrees, SQLite
- Intent-Schicht: `work`, `continue`, `next`, Notion-Sync
- Kosten/Performance: `estimate`, `investigate`, `context`, `cost`, MCP, TUI, Metriken

## In Arbeit (Konsolidierung)

- Architektur-/Sicherheits-/Performance-Audits unter `docs/consolidation/`
- Öffentliche Docs-Site (diese Site) + generierte CLI-Referenz
- OSS-Bereitschaft: Apache 2.0, CONTRIBUTING, Beispiele

## Priorisiert als Nächstes

| Priorität | Punkt |
| --- | --- |
| P0 | Coverage `workflow` / `intent/executor` > 50 % |
| P0 | anbieterexakte Tokenizer |
| P1 | persistenter Kontext-Pack-Cache zwischen Läufen |
| P1 | vollständige GitHub Actions CI (test, lint, release) |
| P2 | interaktive bubbletea-TUI |
| P2 | Vektor-Embeddings RAG |
| P3 | Multi-Repo / Remote-State |
