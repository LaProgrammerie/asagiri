# Roadmap

Maßgebende Reihenfolge und Erläuterungen stehen im **[ROADMAP.md](https://github.com/LaProgrammerie/asagiri/blob/main/ROADMAP.md)** neben dem Code; diese Seite ist eine leicht lesbare Spiegelung für die Dokumentationswebsite.

## Geliefert (V1 bis V3)

Die Kern-CLI-Workflows — etwa `init`, `spec`, `plan`, `dev`, `verify` und `review` mit Worktrees und SQLite — die naturalsprachliche Oberfläche (`work`, `continue`, `next`), Notion-Sync sowie Kostendiagnose und Observability rund um `estimate`, `investigate`, `context`, `cost`, MCP, die TUI und Metriken.

## In Arbeit (Konsolidierung)

Laufende Arbeit umfasst Konsolidierungsaudits unter `docs/consolidation/`, diese öffentliche Docs-Site inklusive generierter CLI-Referenz sowie OSS-Vorbereitung (Apache 2.0, `CONTRIBUTING`, kuratierte Beispiele).
