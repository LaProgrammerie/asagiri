# Produkt-zu-Code Workflow

Empfohlene Sequenz:

```bash
asa prototype create "Workspace SaaS mit Team-Einladungen" --product workspace-saas
asa prototype run workspace-saas
asa flows extract workspace-saas
asa flows inspect workspace-saas
asa flows review workspace-saas
asa contracts extract workspace-saas
asa architecture derive workspace-saas
asa spec generate-from-product workspace-saas
asa product review workspace-saas
asa work "develop workspace-saas" --plan-only
```

## Outputs

- klickbarer lokaler Prototyp;
- Flow/Screen-YAML;
- business-aware Flow-Review (Metriken/Observability/Security);
- technische Vertraege;
- Architektur-Projektion (API/Async/Security/Infra);
- generierte Specs/Tasks;
- ausfuehrbarer Work-Plan.
