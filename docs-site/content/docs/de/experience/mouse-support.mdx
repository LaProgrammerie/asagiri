# Maus-Support

Aktivierung:

```yaml
ui:
  mouse: true
```

## Aktuelle Interaktionen

- Pointer-Events in kompatiblen Terminals;
- Resize von Splits per Mausrad (basic);
- Divider-Drag fuer Panel-Resize (basic).

Alle Kernaktionen bleiben vollstaendig ueber Tastatur verfuegbar.
