# Mausunterstützung

Aktivierung:

```yaml
ui:
  mouse: true
```

## Interaktionen (Lot 7D)

- **Linksklick** — Listenzeile wählen (Explorer, empfohlene Aktionen in Mission Control)
- **Doppelklick** — Knoten-/Schritt-Detail öffnen oder von Events springen
- **Rechtsklick** — Kontextmenü (Explain, Graph fortsetzen, Prototype-Pipeline, Trust prüfen…)
- **Hover** — Zeilen-Highlight bei unterstütztem Cell Motion
- **Rad / Trennlinie ziehen** — Split-Panels skalieren

Tastatur bleibt Pflicht: alle Aktionen auch über Shortcuts und **Strg+P**.
