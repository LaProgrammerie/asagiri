# Mission Control

Mission Control starten:

```bash
asa
# oder
asa mission
```

## Inhalt

- Workspace und Branch;
- Runtime- und Session-Status;
- Fortschritt des aktiven Flows;
- aktive Agenten;
- Trust-Dimensionen und Warnungen;
- aktuelle Events.

Mission Control ist der Startpunkt, bevor du in einen spezialisierten Explorer wechselst.
