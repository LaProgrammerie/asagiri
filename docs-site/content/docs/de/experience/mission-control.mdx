# Mission Control

Mission Control öffnen:

```bash
asa
# oder
asa mission
```

## Hauptinhalt

- Workspace und Branch
- Runtime- und Session-Status
- aktive Flow-Schritte mit Fortschritts-Glyphen
- Agent-Theatre-Kurzüberblick
- Trust-Dimensionen und Gesamtscore
- **Kosten heute / Monat** (aus Runtime-Events)
- letzte Events (Live-Feed)
- **empfohlene Aktionen** — priorisierte nächste Schritte mit CLI-Äquivalent (`GetRecommendedActionsQuery`)

Empfohlene Aktionen oder **Strg+P** für Work, Trust-Verify, Graph-Resume oder Prototype-Pipeline.
