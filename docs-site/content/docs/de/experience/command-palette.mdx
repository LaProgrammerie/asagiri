# Command Palette

Globaler Shortcut:

```text
Ctrl+P
```

## Funktionen

- schnelle Suche ueber zentrale Aktionen;
- Navigation zwischen Kern-Screens;
- Ausfuehrung von Command-Bus-Aktionen;
- sichtbare CLI-Aequivalente fuer Reproduzierbarkeit.

## Safety UX

Destruktive Aktionen zeigen eine Bestaetigung mit Impact, CLI-Befehl, Rollback-Hinweis und explizitem Confirm.
