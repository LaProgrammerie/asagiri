# Themes

Verfuegbare Themes:

- `asagiri-dark`
- `asagiri-light`
- `minimal`
- `high-contrast`
- `cyber`

Konfiguration:

```yaml
ui:
  theme: asagiri-dark
```

## Accessibility-Hinweis

`high-contrast` ist die empfohlene Wahl bei niedriger Sichtbarkeit oder schwacher Terminal-Darstellung.
