# Tastatur-Shortcuts

- `Ctrl+P` command palette
- `Ctrl+M` mission control
- `Ctrl+D` dashboard
- `Ctrl+G` graph explorer
- `Ctrl+F` flow explorer
- `Ctrl+K` knowledge explorer
- `Ctrl+T` trust explorer
- `Ctrl+E` explain
- `Ctrl+R` replay explorer
- `Ctrl+L` logs
- `Tab` / `Shift+Tab` Fokus naechstes/vorheriges Panel
- `?` Hilfe-Screen fuer Accessibility und Tasten
- `Esc` Modal oder Palette schliessen
- `q` beenden

Keyboard bleibt die primaere Bedienung, Maus ist optional.
