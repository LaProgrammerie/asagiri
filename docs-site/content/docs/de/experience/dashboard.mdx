# Dashboard

```bash
asa dashboard
```

## Widgets

- Runtime
- Agents
- Trust
- Costs
- Flow
- Events
- Progress

## Responsives Verhalten

- Compact (<100 Spalten): reduzierte Widget-Auswahl in einer Spalte.
- Wide (>=120 Spalten): zweispaltiges Layout.
- Ultra-wide (>=180 Spalten): dreispaltiges Layout.

Refresh ist gedrosselt, um Flicker und CPU-Last zu begrenzen.
