# `asa graph status`

Displays the current state of a graph stored under `.asagiri/graphs/<graph-id>/` (`spec-my-C` §5.4).

## Usage

```bash
asa graph status graph-2026-05-29-9307e4be
asa graph status graph-2026-05-29-9307e4be --json
```

## Output

Terminal mode prints graph id, product, flow, status, node/edge/checkpoint counts, estimated cost, and highest risk.

With `--json`, returns the full graph document plus estimate snapshot.
