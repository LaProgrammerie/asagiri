# `asa graph resume`

Continues a graph that was paused, failed, or interrupted, using persisted checkpoints (`spec-my-C` §5.5, §15).

## Usage

```bash
asa graph resume graph-2026-05-29-9307e4be
asa graph resume graph-2026-05-29-9307e4be --json
```

## Behaviour

Loads `execution-graph.yaml` from `.asagiri/graphs/<graph-id>/`, restores node states from checkpoints, and runs remaining ready nodes. Emits `graph.*` runtime events during resume.
