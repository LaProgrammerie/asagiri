# `asa memory doctor`

Runs memory health checks (phase finale **PF-A-01**): embedder reachability, dimension consistency, orphan entries.

## Usage

```bash
asa memory doctor
```

## What it checks

| Check | When |
| --- | --- |
| **ollama** | `runtime.memory.embedder: ollama` — API reachable |
| **dimensions** | Stored vectors match embedder output size |
| **orphans** | Entries without valid embedding metadata |
