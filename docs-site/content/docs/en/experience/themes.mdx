# Themes

Supported themes:

- `asagiri-dark`
- `asagiri-light`
- `minimal`
- `high-contrast`
- `cyber`

Configuration:

```yaml
ui:
  theme: asagiri-dark
```

## High-contrast guidance

Use `high-contrast` in low-vision contexts or low-quality terminals. The UI increases visual separation and keeps semantic status colors clear.
