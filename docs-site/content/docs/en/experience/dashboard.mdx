# Dashboard

Open the live dashboard with:

```bash
asa dashboard
```

## Widget groups

- Runtime
- Agents
- Trust
- Costs
- Flow
- Events
- Progress

## Responsive behavior

- Compact (<100 columns): focused widget set in a single column.
- Wide (>=120 columns): two-column layout.
- Ultra-wide (>=180 columns): three-column layout.

Dashboard refresh is throttled to avoid flicker and keep CPU usage under control.
