# Mouse Support

Mouse support is enabled with:

```yaml
ui:
  mouse: true
```

## Current interactions

- pointer events in compatible terminals;
- wheel-based split resize (basic);
- divider drag for panel split resize (basic).

Keyboard-first navigation remains mandatory, so all core actions are still available via shortcuts and palette.
