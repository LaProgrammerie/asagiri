# Mouse Support

Enable with:

```yaml
ui:
  mouse: true
```

## Interactions (lot 7D)

- **Left click** — select a list row (explorers, recommended actions on Mission Control)
- **Double click** — open node/step detail or jump from events
- **Right click** — context menu (explain, resume graph, prototype pipeline, trust verify…)
- **Hover** — row highlight when the terminal supports cell motion
- **Wheel / divider drag** — resize split panes (runtime/trust columns on Mission Control)

Keyboard navigation remains mandatory: every action is also available via shortcuts and **Ctrl+P**.
