# Keyboard Shortcuts

Global shortcuts:

- `Ctrl+P` command palette
- `Ctrl+M` mission control
- `Ctrl+D` dashboard
- `Ctrl+G` graph explorer
- `Ctrl+F` flow explorer
- `Ctrl+K` knowledge explorer
- `Ctrl+T` trust explorer
- `Ctrl+E` explain
- `Ctrl+R` replay explorer
- `Ctrl+L` logs
- `Tab` / `Shift+Tab` next/previous pane focus
- `?` accessibility and key help screen
- `Esc` close modal or palette
- `q` quit

Use keyboard navigation as the primary interaction model. Mouse support is additive.
