# Mission Control

Mission Control is the default interactive screen:

```bash
asa
# or
asa mission
```

## What you see

- workspace and branch;
- runtime status and session state;
- active flow progress;
- active agents summary;
- trust dimensions and warnings;
- recent events feed.

## Why it matters

Mission Control is optimized for a quick decision loop: detect issues, open a focused explorer, then trigger the matching CLI action.
