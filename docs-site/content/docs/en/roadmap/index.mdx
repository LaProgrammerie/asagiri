# Roadmap

The authoritative ordering and narrative live in **[ROADMAP.md](https://github.com/LaProgrammerie/hyper-fast-builder/blob/main/ROADMAP.md)** beside the code; this page is a reader-friendly mirror for the docs site.

## Delivered (V1 → V3)

Shipped work spans core CLI workflows — `init`, `spec`, `plan`, `dev`, `verify`, and `review` backed by worktrees and SQLite — plus the natural-language intent surface (`work`, `continue`, `next`) with Notion sync, and the cost or observability stack around `estimate`, `investigate`, `context`, `cost`, MCP, the TUI, and metrics.

## In progress (consolidation)

Active housekeeping focuses on consolidation audits under `docs/consolidation/`, landing the public docs site you are reading with its generated CLI reference, and OSS readiness items such as Apache 2.0 licensing, `CONTRIBUTING`, and curated examples.
