# Roadmap

Canonical roadmap: **[ROADMAP.md](https://github.com/LaProgrammerie/hyper-fast-builder/blob/main/ROADMAP.md)** at the repository root.

## Delivered (V1 → V3)

- CLI primitives: `init`, `spec`, `plan`, `dev`, `verify`, `review`, worktrees, SQLite
- Intent layer: `work`, `continue`, `next`, Notion sync
- Cost/perf: `estimate`, `investigate`, `context`, `cost`, MCP, TUI, metrics

## In progress (consolidation)

- Architecture/security/performance audits under `docs/consolidation/`
- Public docs site (this site) + generated CLI reference
- OSS readiness: Apache 2.0, CONTRIBUTING, examples
