# Raccourcis clavier

- `Ctrl+P` command palette
- `Ctrl+M` mission control
- `Ctrl+D` dashboard
- `Ctrl+G` graph explorer
- `Ctrl+F` flow explorer
- `Ctrl+K` knowledge explorer
- `Ctrl+T` trust explorer
- `Ctrl+E` explain
- `Ctrl+R` replay explorer
- `Ctrl+L` logs
- `Tab` / `Shift+Tab` focus panneau suivant/précédent
- `?` écran aide accessibilité + touches
- `Esc` fermer modal ou palette
- `q` quitter

Le clavier reste l’interface de référence. La souris est un complément.
