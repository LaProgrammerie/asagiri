# Dashboard

```bash
asa dashboard
```

## Widgets

- Runtime
- Agents
- Trust
- Costs
- Flow
- Events
- Progress

## Responsive

- Compact (<100 colonnes) : jeu réduit de widgets en colonne unique.
- Wide (>=120 colonnes) : disposition en deux colonnes.
- Ultra-wide (>=180 colonnes) : disposition en trois colonnes.

Le refresh est throttlé pour limiter le flicker et la charge CPU.
