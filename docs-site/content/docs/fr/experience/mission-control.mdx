# Mission Control

Ouvrir Mission Control :

```bash
asa
# ou
asa mission
```

## Contenu principal

- workspace et branche ;
- statut runtime et session ;
- progression du flow actif ;
- synthèse des agents actifs ;
- dimensions trust et warnings ;
- événements récents.

Mission Control sert de point d’entrée pour diagnostiquer puis basculer vers un explorer ciblé.
