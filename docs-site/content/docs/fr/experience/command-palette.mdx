# Command Palette

Raccourci global :

```text
Ctrl+P
```

## Fonctions

- recherche rapide d’actions ;
- navigation immédiate vers les écrans clés ;
- déclenchement d’actions via CommandBus ;
- affichage des équivalents CLI.

## Sécurité UX

Les actions destructives ouvrent une confirmation avec impact, commande CLI, rollback et validation explicite.
