# Thèmes

Thèmes supportés :

- `asagiri-dark`
- `asagiri-light`
- `minimal`
- `high-contrast`
- `cyber`

Configuration :

```yaml
ui:
  theme: asagiri-dark
```

## Recommandation accessibilité

Utiliser `high-contrast` pour maximiser la lisibilité en contexte basse vision ou terminal de faible qualité.
