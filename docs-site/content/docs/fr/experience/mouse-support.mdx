# Support souris

Activation :

```yaml
ui:
  mouse: true
```

## Interactions (lot 7D)

- **Clic gauche** — sélection d’une ligne (explorers, actions recommandées sur Mission Control)
- **Double-clic** — ouvrir le détail nœud/étape ou sauter depuis les événements
- **Clic droit** — menu contextuel (explain, reprise graph, pipeline prototype, verify trust…)
- **Survol** — surbrillance de ligne si le terminal supporte le cell motion
- **Molette / drag du séparateur** — redimensionner les panneaux split

Le clavier reste obligatoire : chaque action est aussi disponible via raccourcis et **Ctrl+P**.
