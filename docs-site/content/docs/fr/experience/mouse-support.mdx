# Support souris

Activation :

```yaml
ui:
  mouse: true
```

## Interactions actuelles

- événements pointeur si terminal compatible ;
- resize de split à la molette (basique) ;
- drag du divider pour resize de panneaux (basique).

Toutes les actions critiques restent accessibles au clavier.
