# `asa ready`

Alias de `asa onboard --check-only`. Retourne un rapport readiness JSON ou plain.

## Usage

```bash
asa ready
asa ready --json
asa ready --plain --strict
```

Rapport persisté dans `.asagiri/onboarding/report.json`.
