# Référence CLI générée

Ces pages sont **générées** à partir des définitions Cobra actives. Ne les modifiez pas à la main — elles seront écrasées.

## Régénérer

```bash
go run ./application/cmd/agentflow docs generate-cli \
  --output docs-site/content/docs/en/cli/generated
```

Ou avec un binaire compilé :

```bash
./bin/agentflow docs generate-cli --output docs-site/content/docs/en/cli/generated
```

## Langues

- **Référence complète (anglais)** : [CLI généré — anglais](/docs/cli/generated) — toutes les commandes, tous les drapeaux.
- Cette section **française** propose des pages résumé pour les commandes principales uniquement ; le détail des options reste dans la version anglaise générée.

Parcourir les résumés : ouvrez la section **CLI → généré** dans la barre latérale ou commencez par [l'index généré (EN)](/docs/cli/generated/index).
