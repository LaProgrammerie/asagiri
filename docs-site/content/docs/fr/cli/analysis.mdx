# `asa analysis build`

Couche d'analyse (`spec-my-A` §24.16).

```bash
asa analysis build --product workspace-saas
```

Produit `.asagiri/analysis/<product>/graphs.json` (symbol, dependency, flow, event, api, migration, ownership).

Voir [Couche d'analyse](/docs/fr/concepts/analysis-layer).
