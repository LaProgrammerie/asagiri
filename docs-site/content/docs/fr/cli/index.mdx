# Vue d'ensemble CLI

Binaire d'entrée : **`agentflow`** (`application/cmd/agentflow`).

## Drapeaux globaux

| Drapeau | Description |
| --- | --- |
| `--dry-run` | Simule sans agents externes ni shells de validation |
| (persistant) | Hérité par toutes les sous-commandes |

Environnement : `AGENTFLOW_DRY_RUN=1`, `AGENTFLOW_YES=1` — voir [Variables d'environnement](/docs/fr/reference/environment-variables).

## Groupes de commandes

| Groupe | Commandes |
| --- | --- |
| **Amorçage** | `init`, `doctor`, `version` |
| **Pipeline** | `spec`, `plan`, `enrich`, `dev`, `verify`, `review`, `pr` |
| **Intention** | `work`, `continue`, `next`, `inbox` |
| **Sources** | `sync`, `index` |
| **Coût / contexte** | `estimate`, `investigate`, `context`, `cost`, `cost-report`, `cost-models` |
| **Inspect** | `inspect` (diff, symbol, tests) |
| **Ops** | `status`, `resume`, `report`, `clean` |
| **MCP** | `mcp`, `mcp-serve` |
| **Méta** | `docs`, `help` |

## Référence générée

Les pages par commande (drapeaux synchronisés depuis Cobra) sont sous **[CLI généré](/docs/fr/cli/generated)** (aperçus traduits ; référence complète en anglais sous `/docs/cli/generated`).

Régénérer après modification de la CLI :

```bash
go run ./application/cmd/agentflow docs generate-cli \
  --output docs-site/content/docs/en/cli/generated
```

## Démarrage

```bash
agentflow init
agentflow doctor
agentflow work "my feature" --dry-run --plan-only
```

Voir [Démarrage rapide](/docs/fr/getting-started/quickstart) et [Premier workflow](/docs/fr/getting-started/first-workflow).
