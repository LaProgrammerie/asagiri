# init

CLI : **`asa init`**

## Quand l'utiliser

Les commandes d'amorçage installent ou réconcilient l'arbre de travail Asagiri afin que les nœuds de workflow puissent résoudre de façon fiable la configuration et les métadonnées du dépôt.

## Modes de défaillance

- Cobra rejette les drapeaux inconnus ou les erreurs d'arité avant tout effet de bord sur le dépôt.
- Asagiri ne peut pas résoudre l'amorçage si la commande est invoquée en dehors d'un checkout initialisé.
- L'automatisation en aval peut échouer lorsque les identifiants, les points de terminaison réseau ou les prérequis en cache divergent des attentes.
