# continue

CLI : **`agentflow continue`**

## Quand l'utiliser

Les commandes d'orchestration coordonnent des charges AgentFlow de haut niveau : ingestion, synchronisation, tri de backlog, passes d'investigation, charges contextuelles enrichies, estimations grossières et automatisation spécifique à l'espace de travail.

## Modes de défaillance

- Cobra rejette les drapeaux inconnus ou les erreurs d'arité avant tout effet de bord sur le dépôt.
- AgentFlow ne peut pas résoudre l'amorçage si la commande est invoquée en dehors d'un checkout initialisé.
- L'automatisation en aval peut échouer lorsque les identifiants, les points de terminaison réseau ou les prérequis en cache divergent des attentes.
