# Feuille de route

L’ordre de priorité officiel vit dans **[ROADMAP.md](https://github.com/LaProgrammerie/hyper-fast-builder/blob/main/ROADMAP.md)** aux côtés du code ; cette page en est une vitrine lisible depuis le site.

## Livré (V1 → V3)

La livraison couvre workflows CLI centraux (`init`, `spec`, `plan`, `dev`, `verify`, `review` avec worktrees et SQLite), la surface intention en langage naturel (`work`, `continue`, `next`) ainsi que la synchro Notion, et tout l’échelon observabilité/coût (`estimate`, `investigate`, `context`, `cost`), MCP, TUI ou métriques.

## En cours (consolidation)

Le chantier présent passe par audits de consolidation sous `docs/consolidation/`, la mise à l’air du docs site vous étant affichée avec références CLI régénérées, et quelques chantiers OSS (Apache 2.0, `CONTRIBUTING`, exemples cueillis).
