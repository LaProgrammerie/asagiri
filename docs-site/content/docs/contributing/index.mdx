# Contributing

Full guide at the repository root: **[CONTRIBUTING.md](https://github.com/LaProgrammerie/hyper-fast-builder/blob/main/CONTRIBUTING.md)**.

## Quick checklist

1. Read `AGENTS.md` and `docs/ai/context-map.md`
2. Align features with `docs/ai/active/handoff.md` and product specs (`spec-postv123.md`, etc.)
3. `make build && go test -race ./...`
4. Update CLI docs when commands change:

```bash
go run ./application/cmd/agentflow docs generate-cli \
  --output docs-site/content/docs/cli/generated
```

5. `cd docs-site && npm run build` for MDX changes
6. One PR per coherent topic; no secrets in commits

## Documentation

- Public site: `docs-site/content/docs/` (English, Fumadocs)
- Project canon: `docs/ai/`
