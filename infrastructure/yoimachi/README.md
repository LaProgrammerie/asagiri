# Yoimachi — emplacement projet

Place ici ton fichier **`yoimachi.yaml`** (ou un sous-dossier par workspace) une fois la CLI [Yoimachi](https://github.com/LaProgrammerie/yoimachi) installée.

## Références

- Exemples officiels : [github.com/LaProgrammerie/yoimachi/tree/main/examples](https://github.com/LaProgrammerie/yoimachi/tree/main/examples)
- Doc projet : [`docs/ai/02-architecture.md`](../../docs/ai/02-architecture.md)

Le fichier `yoimachi.yaml.example` est un **squelette commenté** : remplace les valeurs par celles validées pour ta version de Yoimachi.
